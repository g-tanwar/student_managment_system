# Student Management System — Project Report

## 1. Problem Statement

Educational institutions face difficulty in managing student data manually. 
This system provides a digital solution to manage students, courses, 
attendance, and marks efficiently.

---

## 2. Features

- Add / Update / Delete Students
- Course Assignment
- Attendance Tracking
- Marks Management
- Report Generation
